# Marco Improta — minimal academic website

This is a minimal multi-page academic website.

## Files

- `index.html` — intro page
- `publications.html` — books and peer-reviewed articles with publication links
- `teaching.html` — official teaching activities as instructor
- `cv.html` — CV download page
- `style.css` — minimalist LaTeX-like styling
- `marco.jpeg` — profile photo
- `Marco_Improta_CV.pdf` — downloadable CV
- `assets/` — backup copies of photo and CV

## Preview in Codespaces

```bash
python3 -m http.server 8000
```

Then open port 8000 from the **Ports** tab.

## Important

Upload the contents of this folder to GitHub, not the zip file itself. The files `marco.jpeg` and `Marco_Improta_CV.pdf` must be visible in the root of the repository.
